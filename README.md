# lab1-fullstack-kali-server-setup
Full-stack server environment setup on Kali Linux including Apache, MariaDB, PHP, phpMyAdmin, WordPress, and SSL configuration.
The goal of this lab is to understand how web servers, databases, and secure communication work together in a real-world environment.


## Objectives
- Configure Linux server environment
- Install and manage Apache web server
- Set up and secure MariaDB database
- Integrate PHP with Apache
- Install and secure phpMyAdmin
- Deploy WordPress CMS
- Configure SSL encryption for secure communication

## Technologies Used
- Kali Linux
- Apache Web Server
- MariaDB
- PHP
- phpMyAdmin
- WordPress
- OpenSSL

## Lab Steps Summary

### 1. Basic Server Setup
Hostname configured and verified using terminal.

### 2. Apache Installation
Installed and verified Apache web server running successfully.

### 3. MariaDB Installation & Security
Installed MariaDB and secured database using `mysql_secure_installation`.

### 4. PHP Integration
Verified PHP installation using `phpinfo()` page in browser.

### 5. phpMyAdmin Setup
Installed phpMyAdmin and secured access using authentication methods.

### 6. WordPress Installation
Configured database and deployed WordPress on local server.

### 7. SSL Configuration
Generated self-signed SSL certificate and enabled HTTPS encryption.
